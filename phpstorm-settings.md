# PhpStorm 高级应用



## 简单的开始

- `keymap scheme => Default`按键配置选择默认就好。
- `IDE theme => Darcula`主题选择暗色系。
- 打开PhpStorm后，点击菜单`Navigate`,关闭`Status Tool`，`Navigation Bar`
- 打开PhpStorm设置(快捷键`cmd+,`),搜索breadcrumb，关闭选项。
- 关闭标签页,右击打开的文件 => Tabs Placement => None.
